✦ How to Use Mimic Token Downloader

  • Paste your decklist into the text box. Most formats work — MTGO, Moxfield, plain card names, quantities optional.
  • Click Get Tokens From Decklist to fetch all tokens your deck can produce.
  • Click an image to select which art version of each token you want. A purple outline shows your selection.
  • If a token art has multiple set printings, an Alt Prints button appears beneath it. Click it to expand and choose a specific printing.
  • Use Load More under any token group to reveal additional unique artworks beyond the first batch.
  • Optionally name your ZIP in the Deck Name field, then click Download Selected Tokens.
  • Each token downloads as a 320×480 RGB565 .bin file — ready for Mimic devices.
  • Use ↺ Restart to clear everything and start fresh.
⚠️ Download All MTG Tokens fetches every unique token artwork in Magic's history. This may take several minutes and produce a very large ZIP. Use with care!
